\# Nearest Shelter Routing System



A geospatial routing application that identifies the nearest shelter

based on road-network distance.



\## Core technologies



\- Python

\- Streamlit

\- PostgreSQL

\- PostGIS

\- pgRouting

\- Git and GitHub



\## Core MVP



The application allows a user to select a starting location, identifies

the nearest shelter through the road network, calculates the shortest

route using pgRouting, and displays the result on an interactive

Streamlit map.
